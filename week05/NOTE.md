CSS specificity 计算 & 优先级
一个选择器对包含的所有简单选择器计数

[0, 0, 0, 0] => ['inline style attribute', 'ID', 'class attribute and pseudo-class', 'tagName']
举例: #id div.a#id 
这个复合选择器中一共有四个简单选择器: 
* 2个 id 选择器(#id)
* 1个 class 选择器 (.a)
* 1个 tagName 选择器 (div)

[0, 2, 1, 1]
结果是 S = 0*N^3 + 2*N^2 + 1*N^1 + 1*N^1
这里N取一个非常大的值: 1000000
S = 2000001000001

老 IE6 中 N 取值 255

练习:
* div#a.b .c[id=x]  [0, 1, 3, 1]
* #a:not(#b)        [0, 2, 0, 0]
* *.a               [0, 0, 1, 1]
* div.a             [0, 0, 1, 1]
